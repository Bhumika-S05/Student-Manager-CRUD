# 🎓 Student Manager CRUD App

A full-stack CRUD application using Node.js, Express, MongoDB, and a user-friendly frontend.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## ▶️ How to Run

1. Install dependencies:
   npm install

2. Start backend:
   node backend/server.js

3. Open frontend:
   Open frontend/index.html in browser
